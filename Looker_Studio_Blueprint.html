Implementation Guide — Looker Studio Setup
Step-by-step Setup
Step 1: Tạo Data Sources trong Looker Studio
Kết nối BigQuery → Project ID → Dataset game_marts

Tạo 7 data sources tương ứng 7 bảng mart. Mỗi source chỉ cần SELECT * (data đã pre-aggregated).

Lưu ý: mart_daily_kpis, mart_session_stats, mart_retention_cohorts có field date → set làm Date Range dimension để Global Filter hoạt động.
Step 2: Tạo Calculated Fields phổ biến
session_duration_min = avg_session_duration_sec / 60
engagement_min = avg_engagement_sec / 60
engagement_ratio = avg_engagement_sec / avg_session_duration_sec
revenue_per_session = total_revenue / total_sessions
retention_d1_pct = d1_retention * 100 (nếu cần hiển thị %)
Step 3: Thiết lập Global Controls
Date Range Control: Thêm ở header, set "Auto date range" = đúng field date của mỗi data source.
Platform Filter: Drop-down control, dimension = platform, áp dụng cross-page.
Compare Period: Bật "Comparison date range" = Previous Period trên từng chart cần so sánh.

Tip: Dùng Report-level filter (không phải page-level) để filter xuyên suốt 5 pages.
Step 4: Theme & Navigation
Theme: Dark mode → consistent với data product. Hoặc dùng brand colors của Flood-It.
Navigation: Thêm page navigation bar ở top mỗi page (buttons link sang 5 pages).
Mobile: Looker Studio hỗ trợ responsive — test trên mobile view vì PO thường check on-the-go.
