🎨 Style Settings (tab STYLE trong Looker Studio)

Bar Colors (Series Colors)

Vì dùng severity làm Breakdown Dimension, Looker sẽ tạo 3 series tự động.

Click vào mỗi series → đổi màu:
• severe → #f87171 (đỏ)
• warning → #fbbf24 (cam)
• normal → #60a5fa (xanh, opacity 60%)

Reference Line

Tab Style → Reference Lines → Add:

• Type: Constant Line
• Value: nhập giá trị avg_drop_rate_pct (check trong data)
• Label: "Avg Completion Rate"
• Style: dashed, color #fbbf24

Hoặc nếu muốn dynamic: dùng metric reference line = AVG(completion_rate_pct)

Axes

Y-Axis:
• Label: "Completion Rate (%)"
• Range: 0 → 100
• Grid lines: ON, nhẹ

X-Axis:
• Label: "Level Number"
• Show all labels (hoặc interval=5 nếu nhiều levels)

Bar Spacing & Size

• Bar gap: nhỏ (để bars sát nhau → thấy rõ funnel shape)
• Border radius: nhẹ (rounded top)
• Data labels: OFF (chart sẽ quá rối nếu bật)
• Legend: hiển thị ở bottom hoặc top-right
• Background: trắng hoặc dark tùy theme
