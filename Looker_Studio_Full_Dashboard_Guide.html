3. Global Setup: Range Filter + Navigation Tabs

3A. Tạo Range Filter (Levels: 1 — 50)

1 Tạo 2 Parameters

Resource → Manage parameters → Add a parameter

SettingParameter 1Parameter 2
Parameter namelevel_fromlevel_to
Data typeNumberNumber
Default value150
2 Tạo Calculated Field filter

Vào Resource → Manage added data sources → click EDIT trên data source → Add a field:

Field name:  level_in_range

Formula:
CASE
  WHEN level_number >= level_from
   AND level_number <= level_to
  THEN "yes"
  ELSE "no"
END
3 Thêm 2 Controls lên page

Add a control → Drop-down list

PropertyControl 1 (From)Control 2 (To)
Control fieldlevel_from (parameter)level_to (parameter)
PositionBên tráiBên phải

Thêm Text element "Levels:" phía trước và "—" ở giữa 2 controls.

4 Apply filter cho EVERY chart

Với mỗi chart/table trên dashboard:

Click chart → tab Setup → Add a filter →

Filter name:  Level Range Filter
Condition:    Include  |  level_in_range  |  Equal to (=)  |  yes
💡 Tip: Sau khi setup xong page 1, dùng Ctrl+C / Ctrl+V copy toàn bộ controls (filter + nav tabs) sang page 2 và 3 — đặt cùng vị trí để tạo trải nghiệm nhất quán.

SettingValue

⚠️ Lưu ý Reference Lines:
Looker Studio free tier chỉ hỗ trợ Constant value hoặc metric-based reference lines. Nếu muốn dùng metric-based:
→ Chọn Type = Metric → chọn field avg_drop_rate_pct → Aggregation = MAX.
Đối với 2× AVG, cần tạo thêm calculated field trong BigQuery view: 
-- Thêm vào view
ROUND(avg_drop_rate * 200, 2) AS severe_threshold_pct
Hoặc đơn giản nhất: nhập constant value trực tiếp.

6. Page 3 — Overlay (Drop + Attempts)

Mục tiêu

Chart kết hợp bar (drop rate) và line (avg attempts) trên cùng 1 chart — dùng Combo Chart với dual Y-axis. Giúp check correlation: level có drop rate cao có phải vì quá nhiều attempts không?

Bước 1: Add Combo Chart

1 Add a chart → chọn "Combo chart"

(Đây là chart type cho phép mix bars + lines trên cùng 1 chart)

Bước 2: Tab Setup — Field Mapping

2 Cấu hình fields
SlotFieldGhi chú
Dimension level_number Trục X
Breakdown Dimension severity Tô màu bars theo severity
Metric (Bars) drop_rate_pct_display Aggregation = MAX → hiển thị dạng Bars
Metric (Line) avg_attempts Aggregation = MAX → hiển thị dạng Line
Sort level_number Ascending
Rows per page 100
⚠️ Cách set metric làm Bar vs Line:
Trong Combo chart, Looker Studio chia metrics thành 2 nhóm:
→ "Metric" slot (phía trên) = hiển thị dạng Bars
→ Kéo metric thứ 2 vào slot bên dưới hoặc trong tab Style → mục "Series" → click vào avg_attempts → chọn "Line" thay vì "Bar".

Nếu không thấy option này: vào tab Style → Series #2 → Type: Line
